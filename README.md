# LanternFox Games Website

Static studio website for LanternFox Games, including:
- Landing page for the studio and Double Up
- Privacy Policy page
- Terms of Service page

## Local preview

Open `index.html` directly in a browser, or serve the folder with any static server.

## GitHub Pages deployment

1. Push this repository to GitHub.
2. Open repository **Settings** -> **Pages**.
3. Under **Build and deployment**, set:
   - **Source**: Deploy from a branch
   - **Branch**: `main`
   - **Folder**: `/ (root)`
4. Save and wait for deployment.

Your site URL will be:
`https://<your-github-username>.github.io/<repository-name>/`

## Before launch checklist

- Replace `legal@lanternfoxgames.com (placeholder)` with your real contact email.
- Add exact third-party SDK/provider names in `privacy.html`.
- Review legal text with qualified counsel for your specific jurisdiction needs.
